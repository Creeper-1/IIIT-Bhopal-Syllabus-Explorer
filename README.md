# IIIT Bhopal Syllabus Explorer

A Streamlit application that displays IIIT Bhopal course modules in a
clean, expandable interface.

## Features

-   Password-protected access
-   Subject selection
-   Automatically fetches syllabus from the IIIT Bhopal website
-   Expandable module view
-   Simple and responsive interface

## Technologies Used

-   Python
-   Streamlit
-   BeautifulSoup4
-   Requests
-   lxml

## Installation

``` bash
pip install -r requirements.txt
```

## Run Locally

``` bash
python -m streamlit run app.py
```
